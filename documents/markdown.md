# Markdown Cheatsheet

## Headers

# H1
## H2
### H3
#### H4
##### H5
###### H6

## Emphasis

*Italic*
**Bold**
***Bold and Italic***
~~Strikethrough~~

## Lists

### Unordered
- Item 1
- Item 2
  - Subitem 2.1
  - Subitem 2.2

### Ordered
1. First item
2. Second item
3. Third item

## Links

[Link text](https://www.example.com)

## Images

![Alt text](https://example.com/image.jpg)

## Blockquotes

> This is a blockquote.
> It can span multiple lines.

## Code

Inline code: `code`

Code block:
```python
def hello_world():
    print("Hello, World!")
```

- [ ] Uncompleted task
